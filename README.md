## Run Locally

Clone the project

```bash
  git clone https://github.com/kalai-senthil/remainderapp-using-tauri-solidjs
```

Go to the project directory

```bash
  cd remainderapp-using-tauri-solidjs
```

Install dependencies

```bash
  npm install
```

Start the server in dev mode

```bash
  npm run tauri:dev
```
